Anleitung

So verwendest du diese Plattform:

  1. Wähle ein Kapitel aus dem Menü links
  2. Lies die Erklärungen und Beispiele
  3. Schreibe und bearbeite Python-Code in den Editoren
  4. Klicke auf "Code ausführen", um den Code auszuführen
  5. Verwende "Zurücksetzen", um den Code auf den Ausgangszustand zurückzusetzen
  6. Markiere Kapitel als abgeschlossen, um deinen Fortschritt zu verfolgen

Tipp:

Du kannst die Tastenkombination Strg + Enter verwenden, um Code schnell auszuführen.
